# Credit-Card-Transaction-Analysis
“A professional Power BI dashboard analyzing credit card transactions with KPIs, revenue trends, customer insights, card performance, and DAX-based analytics
📌 Key Features

KPIs: Total Revenue, Total Transactions, Total Interest, Transaction Count
Customer Insights: Revenue by Age Group, Education Level, Job, State
Card Performance: Revenue by Card Category & Card Company
Expenditure Analysis: Revenue by Expenditure Type, Used Chip
Trend Analysis: Quarterly & Monthly Revenue
Clean Corporate UI: Professional theme, consistent visuals, and modern layout
Dynamic Measures: WOW, YTD, Percentage contribution, Age Grouping etc.

## Key Decisions & Tradeoffs
- Used a public Kaggle credit card transaction dataset for portfolio learning purposes
- Chose DAX measures for trend analysis (WOW, YTD) instead of pre-aggregating in SQL, to keep the dashboard dynamic and interactive for end users
- Segmented customers by category, education, and employment type to surface the most actionable revenue drivers rather than reporting raw totals alone

## Limitations
- Dataset is static and does not reflect real-time transaction data
- Customer segmentation is based on available demographic fields only; deeper behavioral segmentation would require additional data
